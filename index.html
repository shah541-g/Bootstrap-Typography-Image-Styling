15) Inline html Tags for text

  1. em: italic text
  2. del: perform same rule as of text-line-through utility class
  3. small: unimportant text
  4. strong: important text
  5. lead: important text
  6. s: perform same rule as of text-line-through utility class
  7. ins: perform same act as text-underline
  8. u: perform same act as text-underline
all of them have bootstrap utility classes:
  1. em: fst-italic
  2. del: text-decoration-line-through
  3. small: small
  4. strong: fw-bold
  5. lead: lead
  6. s: text-decoration-line-through
  7. ins: text-decoration-underline
  8. u: text-decoration-underline

16) BlockQuote

Blockquote means quoting a sentence, or a passage. main classes used in bootstrap for this purpose are: blockquote and blockquote-footer
